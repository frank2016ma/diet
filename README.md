# 力量训练基础

## 深蹲

- 脚跟与肩同宽(可以略宽些)
- 脚尖外展30°(两脚呈等边三角形)
- 膝关节外展
- **背部前倾45°**
- 双眼注视前方地面(脚前一米到一米五的位置, **不要向上看**)
- 重心不要前移到脚尖
- 双手处于杠铃之上(大拇指要位于杠铃杆上方)
- 杠铃杆在架子上的高度要低于斜方肌
- 肘部上抬, 胸部挺起
- 杠铃杆压在肩胛骨下方一点
